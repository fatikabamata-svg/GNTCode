Notre mission

Valoriser les trésors touristiques de la Guinée

Guinea National Tour est conçu comme une passerelle entre les voyageurs, les destinations locales et une base de données MySQL fiable. Cette première vitrine pose les fondations visuelles du futur site web et de l'application mobile.

Base MySQL Catégories, régions, destinations, images, utilisateurs et réservations.
API REST Routes prévues pour lister, détailler et réserver les destinations.
Contenu évolutif 23 destinations au lancement, avec possibilité d'ajouter de nouveaux sites.
